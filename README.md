# Customer Retention Cohort Analysis

This project analyzes customer retention using cohort analysis on transactional sales data.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Jupyter Notebook
- DAX

## Project Overview
- Customers are grouped into monthly cohorts based on their first purchase.
- Retention percentages are calculated across subsequent months.
- A cohort heatmap is created using conditional formatting in Power BI.
- Interactive tooltips display cohort size and retention KPIs.

## Files
- `01_data_cleaning.ipynb`: Data cleaning and cohort preparation
- `data/`: Raw and processed datasets

## Key Insights
- Early-month churn is highest across most cohorts.
- Certain cohorts show stronger long-term retention.
- Heatmap visualization makes retention trends easy to compare.
